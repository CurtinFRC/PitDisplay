# Pit Display

A tool to display information on a screen in the pits.

## Setup

To setup the project, first clone it:

```bash
git clone https://github.com/CurtinFRC/PitDisplay.git
```

Then install dependancies with pnpm:

```bash
pnpm install
```

Then run with:

```
pnpm run dev
```

## Building

To build locally use:

```bash
pnpm run build
```
